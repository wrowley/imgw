# imgw
Simple C image processing library with a permissive license.

The functionality for each filetype builds separately.

| File type     | Reads              | Writes             |
| ------------- |      :---:         |        :---:       |
| BMP (8-bit)   | :x:                | :heavy_check_mark: |
| BMP (24-bit)  | :x:                | :heavy_check_mark: |
